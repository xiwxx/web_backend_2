Лабораторная работа №2

Базалеева Карина

0. Развертывание ресурсов

Файлы проекта (скрипты, изображения, архивы) загружены на удаленный сервер. Проверка доступности подтвердила корректность конфигурации окружения.

1. Запрос главной страницы (HTTP/1.0)

Выполнен базовый GET-запрос. Использован протокол версии 1.0 через терминальное соединение.

2. Запрос index.php (HTTP/1.1)

Запрос по протоколу HTTP/1.1. В заголовках обязательно передан Host для идентификации ресурса.

3. Получение размера архива

С помощью метода HEAD определен размер файла file.tar.gz (поле Content-Length) без загрузки самого тела файла.

4. Проверка типа изображения

Метод HEAD позволил определить MIME-тип файла image.png через заголовок Content-Type.

5. POST-запрос с данными

Имитация отправки формы. Передан текстовый комментарий, сервер успешно принял и подтвердил обработку данных.

6. Частичный запрос контента (Range)

Запрошены первые 100 байт архива. Использован заголовок Range, сервер вернул статус 206 Partial Content.

7. Определение кодировки документа

Анализ заголовков ответа для поиска параметра charset, определяющего кодировку страницы.
